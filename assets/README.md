# Assets Folder

This folder contains all media and document assets for the portfolio.

## Directory Structure

```
/assets
├── /images
│   ├── profile-photo.jpg         (Your professional photo)
│   ├── project-swarm-robots.jpg  (Demo image for swarm robots project)
│   ├── project-quadruped.jpg     (Demo image for quadruped project)
│   ├── project-mppi-navigation.jpg (Demo image for MPPI navigation)
│   ├── project-vision-navigation.jpg (Demo image for vision navigation)
│   └── project-wearable.jpg      (Demo image for wearable system)
└── /documents
    └── resume.pdf                (Your resume PDF)
```

## How to Add Your Assets

1. **Profile Photo**: Add your professional photo as `profile-photo.jpg` in `/assets/images/`
2. **Project Images**: Add your project demo images/screenshots in `/assets/images/`
3. **Resume**: Add your resume PDF as `resume.pdf` in `/assets/documents/`

## Recommended Image Specifications

- **Profile Photo**: Square format, at least 800x800px
- **Project Images**: Landscape format, at least 1200x800px
- **File formats**: JPG or PNG
- **File size**: Keep under 2MB for optimal loading speed

## Note

If you don't add these files, the portfolio will show placeholder text instead. The site will still function normally.
